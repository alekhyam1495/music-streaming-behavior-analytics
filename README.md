**#Music Streaming Behavior Analysis**

**Overview:**
This project analyzes personal Spotify streaming history data to uncover user listening behaviors, engagement patterns, artist loyalty, skip behavior, and time-based music consumption trends using Python, Pandas, SQL concepts,Tableau and power BI. 
The project focuses on transforming raw streaming logs into meaningful behavioral insights through data cleaning, feature engineering, exploratory data analysis (EDA), and interactive visualizations.

**Project Objectives**
The primary goals of this project are to:
Analyze 10,000+ Spotify streaming records
Identify peak listening hours and listening habits
Discover top artists, songs, and repeated listening behavior
Detect song skip patterns using playback duration
Perform time-series and behavioral analysis
Build interactive dashboards and visual storytelling
Demonstrate end-to-end analytics workflow using Python and Tableau
 
 **Dataset**
The dataset contains Spotify streaming history exported from Spotify account data.

**Key Columns**
Column	Description
endTime	Timestamp when playback ended
artistName	Artist name
trackName	Song name
msPlayed	Milliseconds played
